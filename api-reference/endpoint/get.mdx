### Request Details

- **URL:** `https://api.coinfi.zone/transactions`
- **Method:** `GET`
- **Query Parameter:** `limit` (optional) â€“ defines the maximum number of transactions to return.

### Example Request

```sh
curl -X GET "https://api.coinfi.zone/transactions?limit=10" -H "Accept: application/json"
```
