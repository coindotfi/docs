- **Public Keys:** Identify the wallet owner.
- **Private Keys:** Enable the owner to sign and send messages.
- **Quantum One–Time Pad (QOTP):** A dynamic, color–coded pad solved via entropy-based rotations.

## ENIGMA: ROSARIO-WANG Cipher System Example

### Introduction
This section illustrates how the system integrates cryptographic mechanisms within a holographic manifold using:
- The universal language
- Private key and subkeys
- A secret synonym mapping (holographic morphism)
- The $\Sigma$-Interactive Protocol

### Step 1: Universal Language ($\Lambda$)
<div className="equation">
  $\Lambda = \lbrace \mathrm{A}, \mathrm{B}, \ldots, \mathrm{Z} \rbrace$
</div>

### Step 2: Private Key ($\Omega$)
For example:
<div className="equation">
  $\Omega = \lbrace \mathrm{S}, \mathrm{E}, \mathrm{C}, \mathrm{R}, \mathrm{E}, \mathrm{T} \rbrace$
</div>

### Step 3: Subkeys ($\Omega_i$)
Split as:
<div className="equation">
  $\begin{aligned}
    \Omega_1 &= \mathrm{S} \\
    \Omega_2 &= \mathrm{E} \\
    \Omega_3 &= \mathrm{C} \\
    \Omega_4 &= \mathrm{R} \\
    \Omega_5 &= \mathrm{E} \\
    \Omega_6 &= \mathrm{T}
  \end{aligned}$
</div>

### Step 4: Secret Synonym Mapping ($\Psi$)
Mapping hyperplanes to directions:
- $\Gamma_1$ (Yellow) → $\Xi_1$ (UP)
- $\Gamma_2$ (Green) → $\Xi_2$ (DOWN)
- $\Gamma_3$ (Blue) → $\Xi_3$ (LEFT)
- $\Gamma_4$ (Red) → $\Xi_4$ (RIGHT)

Expressed as:
<div className="equation">
  $\Psi: \Gamma_i \rightarrow \Xi_i$
</div>

### Step 5: Commitment ($\Gamma$)
Combined as:
<div className="equation">
  $\Gamma = H(\Omega \mid H(\Psi))$
</div>

### Step 6: $\Sigma$-Interactive Protocol (Per Subkey)
For each subkey $\Omega_i$, the observer:
1. Randomly distributes a subset $\Pi_i \subset \Lambda$ across hyperplanes.
2. Locates $\Omega_i$ in a hyperplane $\Gamma_j$.
3. Determines the witness $\Xi_j = \Psi(\Gamma_j)$.
4. Computes a commitment:
<div className="equation">
  $\text{Comm}_i = H(\Omega_i \mid \Xi_j \mid r_i)$
</div>
5. Responds to a challenge $e_i$:
<div className="equation">
  $s_i = f(\Omega_i, \Xi_j, e_i, r_i)$
</div>
6. The verifier checks:
<div className="equation">
  $B(\text{Comm}_i, e_i, s_i)$
</div>

### Step 7: Witness Sequence
After all rounds, the observer obtains:
<div className="equation">
  (\Xi_1, \Xi_2, \Xi_3, \Xi_4, \Xi_5, \Xi_6)
</div>
For example:
- $\Xi_1 =$ UP, $\Xi_2 =$ DOWN, $\Xi_3 =$ LEFT, $\Xi_4 =$ RIGHT, $\Xi_5 =$ DOWN, $\Xi_6 =$ UP

### Step 8: Verification
The verifier confirms each response against the commitments and witness values.
